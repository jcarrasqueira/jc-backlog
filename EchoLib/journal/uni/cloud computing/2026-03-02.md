## Business Capabilities
### 1. Searchable Media Catalog
- provides searchable access to cleaned, structured movie/tv shows data (title, year, cast, genre, runtime)
### 2. Rating and Review System
- manages user ratings and reviews for any title, aggregates them into global scores (number of reviews, average rating)
### 3. Personal watchlist
- allows authenticated users to store personal watchlists, which can be properly categorized as *currently watching*, *planning to watch*, *finished watching*
### 4. Recommendation System
- suggests tv shows or movies to users based on their watching and ratings history 
### 5. Trending Analytics
- provides top‑rated, most popular or trending movies and tv shows over time

### 6. User Profile/Account
- account creation 
- user registered has access to their profile with username, profile image, private and public watchlists

## Use Cases
### Search titles
- **description:**  user enters search terms or filters by type (movie or tv show), genre, year range or minimum rating. System returns paginated results showing titles correspondent to the search query.
- **actor:** user
- **goal:** find movies or tv shows matching the user input.
- **business capability:** Searchable Media Catalog
### View title details
- **description:**  user requests details for a title by its id. System returns details of the title such as title, type (tv show or movie), release date, seasons, genres, average rating, main cast, and description (if available).
- **actor:** user
- **goal:** get all the information about a specific movie or tv show
- **business capability:** Searchable Media Catalog
### Rate and review a title
- **description:**  user submits a rating from 1 to 10 and optional text review for a title. System stores the review and recalculates the title's average rating, updating it accordingly.
- **actor:** authenticated user
- **goal:** share personal review and rating to public review system.
- **business capability:** Rating and Review System
