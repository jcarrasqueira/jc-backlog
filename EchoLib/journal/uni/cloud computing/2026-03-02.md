# Project description
With the growing amount of content to explore both online and offline, it has become difficult to keep track of everything we watch. Our project addresses this issue by providing a platform which offers unified catalog of both tv shows and movies,  a public review system, and a personal watchlist to keep users organize all the titles they interact with. On top of this, the platform includes a recommendation system that suggests future watches based on each user's history.

**objective:** build and deploy cloud-native analytics and recommendation platform built on large-scale tv shows and movie datasets

## Business Capabilities
### 1. Searchable Media Catalog
- provides searchable access to cleaned, structured movie/tv shows data (title, year, cast, genre, runtime)
### 2. Rating and Review System
- manages user ratings and reviews for any title, aggregates them into global scores (number of reviews, average rating)
### 3. Personal watchlist
- allows authenticated users to store personal watchlists, which can be properly categorized as *currently watching*, *planning to watch*, *finished watching*
### 4. Recommendation System
- suggests tv shows or movies to users based on their watching and ratings history 
### 5. Trending Analytics
- provides top‑rated, most popular or trending movies and tv shows over time
