## User Service
### user
- table for all users

| column         | type      | postgresql type                               | notes                                                                                                                                         |
| -------------- | --------- | --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **user_id**    | int       | serial                                        | PK (auto increment in db)                                                                                                                     |
| **username**   | str       | varchar(255)                                  | unique and not null                                                                                                                           |
| **password**   | str       | text                                          | optional, since is null for OAuth users. IMP: backend must ensure a user non OAuth has password != null. Also must be a hash of the password. |
| **gender**     | str       | varchar(10)                                   | backend must enforce options: *M,F,NB or O*                                                                                                   |
| **age**        | int       | integer                                       |                                                                                                                                               |
| **created_at** | timestamp | timestamp or datetime (str with no time zone) | ```DEFAULT NOW()```                                                                                                                           |
| **updated_at** | timestamp | timestamp or datetime                         | ```DEFAULT NOW()```                                                                                                                           |
### user_oauth
- table to store users with OAuth2.0 and their tokens
- one‑to‑one → users
- considered that we only will accept OAuth2.0 for google accounts
    - if other providers must add *provider_name* 

| column            | type      | postgresql type       | notes                                                          |
| ----------------- | --------- | --------------------- | -------------------------------------------------------------- |
| **oauth_id**      | int       | serial                | PK (auto increment in db)                                      |
| **user_id**       | int       | integer               | FK to users tables, on delete cascade                          |
| **provider_id**   | str       | varchar(255)          | not null                                                       |
| **refresh_token** | str       | text                  | optional column (if it justifies having it). should be hashed. |
| **expires_at**    | timestamp | timestamp or datetime | when token expires                                             |
| **created_at**    | timestamp | timestamp or datetime | `DEFAULT NOW()`                                                |
| **updated_at**    | timestamp | timestamp or datetime | `DEFAULT NOW()`                                                |
### user_sessions
- table to store user current sessions
- one-to-one -> users
    - on login → delete old session → create new one
    - if supports multiple devices logins then maybe many-to-one (1 user to many sessions)

| column            | type      | postgresql type       | notes                                 |
| ----------------- | --------- | --------------------- | ------------------------------------- |
| **session_id**    | str       | varchar(255) or UUID  | PK                                    |
| **user_id**       | int       | integer               | FK to users tables, on delete cascade |
| **refresh_token** | str       | text                  | Should be hashed, not null            |
| **created_at**    | timestamp | timestamp or datetime | `DEFAULT NOW()`                       |
| **updated_at**    | timestamp | timestamp or datetime | `DEFAULT NOW()`                       |
| **expires_at**    | timestamp | timestamp or datetime | when token expires                    |
| **revoked**       | bool      | boolean               | for jwt to logout                     |

## Movie Catalog
### movies
- table to store movies

| column              | type | postgresql type | notes                          |
| ------------------- | ---- | --------------- | ------------------------------ |

| **movie_title**     | str  | varchar(255)    | not null                       |
| **description**     | str  | text            |                                |
| **imbd_url**        | str  | text            |                                |

## Review System
### ratings
- many‑to‑one → users, many‑to‑one → movies

| column             | type      | postgresql type       | notes                                                          |
| ------------------ | --------- | --------------------- | -------------------------------------------------------------- |
| **rating_id**      | int       | serial                | PK (auto increment in db)                                      |
| **user_id**        | int       | integer               | FK users                                                       |
| **movie_id**       | int       | integer               | FK movies                                                      |
| **rating**         | int       | integer               | `CHECK (rating BETWEEN 1 AND 5)`                               |
| **review**         | str       | text                  |                                                                |
