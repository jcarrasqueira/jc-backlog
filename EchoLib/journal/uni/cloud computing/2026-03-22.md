## User Service
### user
- table for all users

| column         | type      | postgresql type                               | notes                                                                                                                                         |
| -------------- | --------- | --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **user_id**    | int       | serial                                        | PK (auto increment in db)                                                                                                                     |
| **username**   | str       | varchar(255)                                  | unique and not null                                                                                                                           |
| **password**   | str       | text                                          | optional, since is null for OAuth users. IMP: backend must ensure a user non OAuth has password != null. Also must be a hash of the password. |
| **gender**     | str       | varchar(10)                                   | backend must enforce options: *M,F,NB or O*                                                                                                   |
| **age**        | int       | integer                                       |                                                                                                                                               |
| **created_at** | timestamp | timestamp or datetime (str with no time zone) | ```DEFAULT NOW()```                                                                                                                           |
| **updated_at** | timestamp | timestamp or datetime                         | ```DEFAULT NOW()```                                                                                                                           |
### user_oauth
- table to store users with OAuth2.0
- considered that we only will accept OAuth2.0 for google accounts
    - if other providers must add *provider_name* 

| column         | type      | postgresql type                               | notes                                       |
| -------------- | --------- | --------------------------------------------- | ------------------------------------------- |
| **oauth_id**   | int       | serial                                        | PK (auto increment in db)                   |
| **user_id**    | int       | int                                           | FK to users tables, on delete cascade       |
| **username**   | str       | varchar(255)                                  | unique and not null                         |
| **gender**     | str       | varchar(10)                                   | backend must enforce options: *M,F,NB or O* |
| **age**        | int       | integer                                       |                                             |
| **created_at** | timestamp | timestamp or datetime (str with no time zone) | ```DEFAULT NOW()```                         |
| **updated_at** | timestamp | timestamp or datetime                         | ```DEFAULT NOW()```                         |
