#### Crash violation
- having a folder with the same name of the log to be created makes the code crash instead of invalid message
- where it occurs in the code (*log_format.py*):
```python
def check_token(path: str, token: str) -> bytes:
	try:
		with open(path, 'rb') as f:
			header = f.read(HEADER_SIZE)
	except FileNotFoundError:
		raise ValueError("invalid")
(...)
```
in the code check_token attempts to open the log file with `with open(path, 'rb') as f:\` and try except block only checks for FileNotFoundError. When a folder with same name of the log to created is provided the program tries to read it but will trigger IsADirectoryError which is not handled by the try block, resulting in this:

![[Pasted image 20260430152325.png]]

- steps to replicate it:
1. create a folder in base directory of the project: mkdir fake_log (in case of using linux)
2. run the command: python3 logread.py -K secret -S fake_log
